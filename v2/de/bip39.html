BIP39 utility

BIP39-Wort zu Binär

Wandeln Sie ein einzelnes BIP39-Wort in seine 11-Bit-Darstellung um. Geben Sie niemals eine vollständige Wiederherstellungsphrase online ein.

Security note. This tool accepts one word only. Do not type, paste or submit an entire recovery phrase.

11-bit binary

1 = markieren · 0 = leer lassen

How to read it

  1. Enter one word, such as abandon.
  2. Read its eleven bits from left to right.
  3. Mark each 1; leave each 0 empty.
  4. Verify before continuing to the next line.
